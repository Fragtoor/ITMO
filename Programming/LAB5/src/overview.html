Описание проекта

Консольное приложение для управления коллекцией объектов MusicBand. Программа позволяет добавлять, удалять, изменять и просматривать элементы коллекции, а также сохранять и загружать данные из CSV файла.

Команды

Команда Описание
help справка по доступным командам
info информация о коллекции (тип, дата инициализации, количество элементов)
show список всех элементов коллекции
add добавить новый элемент
update id обновить элемент по id
remove_by_id id удалить элемент по id
clear очистить коллекцию
save сохранить коллекцию в файл
execute_script file выполнить скрипт
exit выход
add_if_min добавить новый элемент, если он минимальный
remove_greater удалить все элементы, превышающий заданный
history список последних десяти команд
sum_of_number_of_participants сумма значений поля numberOfParticipants для всех элементов коллекции
average_of_number_of_participants среднее значение поля numberOfParticipants для всех элементов коллекции
filter_contains_name элементы, значение поля name которых содержит заданную подстроку
back n отмена последних n команд

Запуск программы

При запуске программы обязательно в качестве аргумента передать файл с начальными данными.
